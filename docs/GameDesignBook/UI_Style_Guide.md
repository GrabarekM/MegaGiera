# UI Style Guide — v0.1

Dokument opisuje kierunek. Nie definiuje jeszcze gotowych komponentów ani wartości implementacyjnych.

## Main Menu

Prosty punkt wejścia z jednym dominującym działaniem. Continue pokazuje tożsamość aktualnego runu; operacje destrukcyjne wymagają potwierdzenia. Oprawa przypomina ciężką stronę tytułową lub kamienną tablicę.

## Character Creation

Przypomina kartę postaci lub otwartą księgę. Kolejne wybory mają widoczny koszt, pozostałą pulę i podsumowanie wpływu. Gracz zawsze wie, co pozostaje do ukończenia.

## Inventory

Metafora skrzyni lub torby. Lista przedmiotów i szczegóły są oddzielone. Filtry pomagają, ale nie dominują. Akcje destrukcyjne są wyraźnie odseparowane.

## Merchant

Metafora stołu handlowego: oferta kupca i zasoby gracza pozostają wizualnie rozróżnione. Cena, liczba sztuk i skutek transakcji są widoczne przed zatwierdzeniem.

## Dialogue

Rozmowa skupia uwagę na rozmówcy, tekście i wyborach. Niedostępne odpowiedzi wyjaśniają warunek bez ujawniania sekretów. Historia techniczna nie może wypierać aktualnej wypowiedzi.

## Quest

Dziennik przypomina księgę zapisków. Cel bieżący jest krótki i możliwy do śledzenia, a pełna historia pozostaje w osobnym widoku. Status wynika również z tekstu i symbolu, nie wyłącznie z koloru.

## Tooltip

Wyjaśnia obiekt w kontekście. Najpierw nazwa i funkcja, potem wymagania, liczby i źródła modyfikatorów. Tooltip nie zasłania celu i nie jest jedynym miejscem dla informacji krytycznej.

## Window

Jednoznaczny nagłówek, wyraźna hierarchia, widoczne zamknięcie i właściwa obsługa `Escape`. Okna inspirowane księgą, pergaminem, mapą, skrzynią lub sakiewką zachowują wspólne zachowanie.

## Button

Przycisk wygląda jak element możliwy do naciśnięcia. Ma stany normalny, hover, focus, active i disabled. Etykieta opisuje rezultat, a nie ogólne „OK”, jeśli decyzja ma koszt.

## HUD

Pokazuje wyłącznie stan potrzebny podczas eksploracji: postać, zdrowie, najważniejsze zasoby, czas/zagrożenie i skróty. Pozostawia centrum mapy wolne i ogranicza nakładanie paneli.
