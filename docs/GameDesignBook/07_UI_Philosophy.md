# UI Philosophy

Rozdział książki poświęcony zasadzie, że interfejs jest częścią świata, a czytelność ma pierwszeństwo przed dekoracją. Pierwsze rozwinięcie znajduje się w `UI_Philosophy.md`.

## Przyszłe rozdziały

- Hierarchia informacji.
- Nawigacja klawiaturą.
- Zasady modalności.
- Dostępność bez utraty charakteru świata.
